# 🚀 Digital Marketing Course Landing Page

A modern, fully responsive Digital Marketing Course Landing Page built using HTML5, Bootstrap 5, and Custom CSS.

This project showcases a clean UI, responsive layouts, Bootstrap components, and modern landing page design principles.

---

## ✨ Features

- Responsive Design
- Modern Hero Section
- Bootstrap 5 Components
- Sticky Navigation Bar
- Modal Popup Forms
- Lead Generation Form
- Course Highlights Section
- Career Opportunities Cards
- Certification Section
- FAQ Accordion
- Call-to-Action Sections
- Company Partners Section
- Responsive Footer
- Mobile Friendly Layout
- Font Awesome Icons
- Google Fonts Integration

---

## 🛠 Built With

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- Google Fonts
